Image Processing Lab – README
📌 Project Description

This repository contains Digital Image Processing Laboratory Experiments (1 to 11) implemented using Python and standard image processing libraries.
The purpose of this project is to provide practical understanding of fundamental image processing concepts such as image acquisition, enhancement, filtering, restoration, compression, segmentation, and morphological operations.

These experiments help students learn how digital images are represented, processed, analyzed, and improved using computational techniques.

🎯 Objectives
Understand the basics of digital image processing.
Learn practical implementation of image enhancement techniques.
Apply filtering methods in spatial and frequency domains.
Study image restoration and noise reduction methods.
Perform image segmentation and feature extraction.
Explore image compression methods.
Gain hands-on experience with Python-based image processing tools.
🛠️ Technologies & Libraries Used
Python 3.x
OpenCV
NumPy
Matplotlib
SciPy
📚 Experiments Included
1️⃣ Experiment 1 – Image Acquisition and Display

Implemented a program to acquire and display an image.
This experiment demonstrates the working of an image processing system including:

Illumination source
Object reflection
Image sensor
Analog to digital conversion
Digital image generation

Outcome: Understand how real-world images are converted into digital form.

2️⃣ Experiment 2 – Sampling and Quantization

Performed sampling and quantization on an image to observe image quality changes.

Sampling: Reduces spatial resolution
Quantization: Reduces gray-level intensity values

Outcome: Learn effects of resolution loss and color depth reduction.

3️⃣ Experiment 3 – Image Enhancement in Spatial Domain

Implemented enhancement techniques:

Contrast Stretching
Histogram Equalization

These methods improve brightness distribution and image visibility.

Outcome: Learn how to improve poor-quality images.

4️⃣ Experiment 4 – Frequency Domain Filtering

Applied Fourier Transform based filtering:

Low Pass Filter → Removes noise and smooths image
High Pass Filter → Sharpens image and enhances edges

Outcome: Understand frequency domain processing.

5️⃣ Experiment 5 – Image Restoration using Spatial Filters

Added noise to image and restored it using:

Mean Filter
Median Filter

Noise Types:

Gaussian Noise
Salt & Pepper Noise

Outcome: Learn image restoration techniques.

6️⃣ Experiment 6 – Periodic Noise Reduction

Implemented advanced restoration techniques:

Inverse Filtering
Wiener Filtering

Used to remove repetitive noise patterns.

Outcome: Learn degradation model based restoration.

7️⃣ Experiment 7 – Geometric Transformations

Applied transformations such as:

Translation
Rotation
Scaling

Used for resizing and repositioning images.

Outcome: Understand coordinate mapping in images.

8️⃣ Experiment 8 – Color Image Processing and Segmentation

Converted image into HSV color model and segmented based on color ranges.

Applications:

Object detection
Traffic signal recognition
Fruit sorting

Outcome: Learn color-based image analysis.

9️⃣ Experiment 9 – Image Compression

Implemented lossless compression methods:

Huffman Coding
Run Length Encoding (RLE)

Used for reducing file size.

Outcome: Learn data compression concepts.

🔟 Experiment 10 – Edge Detection and Segmentation

Applied Canny Edge Detection to detect boundaries in images.

Applications:

Shape recognition
Object tracking
Medical imaging

Outcome: Learn feature extraction techniques.

1️⃣1️⃣ Experiment 11 – Morphological Operations

Performed operations on binary images:

Dilation
Erosion

Used for removing noise and refining shapes.

Outcome: Learn structural image processing.
